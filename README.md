# certs
ISKCON Certificates

## Installation
1. install [adbhutagaurangadas/proxy](https://github.com/adbhutagaurangadas/proxy) container to separate folder sharing **proxy** network
2. git checkout this repo
3. cp initial_templates templates
4. cp .env.example .env (set your host)
5. chmod 777 public/results
6. docker compose up -d
